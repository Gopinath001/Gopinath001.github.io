The thing i did

So what is touch typing? typing without seeing the keyboards, using the f&j as references because they have bumps on them, you feel them as a reference, You can recognise this by the word  “Type-Writer”,  back in the days they used ti have type writing class everywhere,  even now there are classes but are fewer and fewer nowadays.  Anywhere there are fewer and fewer people trying to type in this way because they have developed their own way of typing. I have taken this task some time before, i would think like i’d learn part of the system and remaining part in upcoming days,  but after this section, i would never go back, maybe i forget that i have to do that or i’ll get something else that I have to do and totally forget about all about it.eventually when i was involved in more and more typing I developed my own typing style which is not perfect but helped me to type faster upto 40 words per minute, which is somewhat good, but i would type with only some of my fingers, which in turn hurts and became tired after somebit. This was not enough and finally i decided I’d give it a try, i know i might go along with this for a few days but eventually will give up on this, so as far as i’m concerned i have to complete this quick, like real quick, So i decided to complete all the course on a single go(with atleast 90% accuracy). So it started, i’ve worked for 8 hours straight & completed everything, i knew how to type now but the problem is practicing, to make it faster and better. The other problem is my old typing habbits began to affect me, the new typing style was slow, thus while typing in my work it started to affect, thus had to go back to old style for completing my work faster, from then on its been a constant struggle between switching between different styles, Everyone should learn this, its super comfy. Besides it’s the official way to type, It’ll be massively helpful if done at a young age, Trust me on this one, Anyhooo0oo I typed this whole post with the new style of typing. One could say that is the reason why it took this long for the post, lol. See you next time!.
